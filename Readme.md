肆意黑是一款以[得意黑](https://github.com/atelier-anchor/smiley-sans)为基础、在笔画风格上进行更加大胆设计的字体。在吸取了得意黑弯钩、折角等设计后进行了更多艺术的处理而衍生出肆意的设计。

由于并非所有字符都含有后续设计所包含的笔画类型，仍保留了大量原得意黑设计字符，西文等部分同样进行保留。
